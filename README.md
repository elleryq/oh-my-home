# oh-my-home
Home stuffs

The script will install:
 * git
 * zsh and oh-my-zsh
 * pyenv
 * git-cola
 * cscope
 * ctags
 * gawk
 * build-essential
 * vim and my forked exvim
 * gcin
 * parcellite
 * terminator

After installation, $HOME will contain these files/directories:
 * .oh-my-zsh
 * .oh-my-home
 * .pyenv
 * .zshenv
 * .zshrc
 * exvim
